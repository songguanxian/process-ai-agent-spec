# 工序管控系统 AI 助手需求与开发规格

本仓库用于管理《工序管控系统 AI 助手（Agent）》需求澄清成果、合并版需求说明书和 Codex AI Coding 开发规格说明。

## 目录

- `outputs/工序AI助手_AI_Coding_开发规格说明.md`：可直接交给 Codex 开展 AI Coding 的开发规格说明。
- `outputs/工序AI助手需求说明书_多领域澄清合并版.docx`：多领域 Grilling 澄清后的需求合并版。
- `outputs/工序AI助手需求说明书_OCR场景细化版.docx`：OCR/视觉识别场景细化版。
- `work/`：生成、合并和校验文档使用的脚本与中间文本；渲染校验图片和 PDF 已通过 `.gitignore` 排除。

## 版本管理说明

仓库保留最终交付文档和可追溯生成脚本，不提交 Word 渲染校验产生的临时图片、PDF 和 Office 临时文件。
